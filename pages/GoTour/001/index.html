# Go语言的下载与安装

  1. 下载。
  2. 去安装。
  3. 执行代码。

按照此处描述的步骤快速下载并安装。

有关安装的其他内容，您可能对以下内容感兴趣：

  • Go安装管理--如何安装多个版本并卸载。
  • 从源代码开始安装——如何检查源代码，在自己的机器上构建并运行它们。

# Go的安装

# Linux系统下的安装

  1. 将下载的压缩文件提取到/usr/local中，在/usr/local/Go中创建Go的树形目录。

重要提示：此步骤将在提取之前删除/usr/local/go上以前的安装（如果有）。请在继续操作之前备份所有数据。

比如，以root用户身份或通过sudo运行以下操作：

rm-rf/usr/local/go&&tar-C/usr/local-xzf go1.14.3.linux-amd64.tar.gz
2. 将/usr/local/go/bin添加到PATH环境变量中。 您可以通过将以下行添加到$HOME/.profile或/etc/profile（用于全系统安装）来完成此操作：

export PATH=$PATH:/usr/local/go/bin

注意：对配置文件所做的更改可能要等到下次登录计算机时才能应用。要立即应用更改，只需直接运行shell命令，或者使用source$HOME/.profile等命令从概要文件中执行这些命令。
3. 通过打开命令提示符并键入以下命令来验证您是否已安装Go：

$go version
4. 确认该命令打印出的已安装的Go版本。

# Mac系统下的安装

  1. 打开您下载的软件包文件，然后按照提示安装Go。

该软件包将Go发行版安装到/usr/local/Go。该包应该将/usr/local/go/bin目录放在PATH环境变量中。您可能需要重新启动任何打开的终端会话才能使更改生效。
2. 通过打开命令提示符并键入以下命令来验证您是否已安装Go：

$go version
3. 确认该命令打印已安装的Go版本。

# windows系统下的安装

  1. 打开您下载的MSI文件，然后按照提示安装Go。

默认情况下，安装程序将安装Go to Program Files或Program Files（x86）。您可以根据需要更改位置。安装后，您需要关闭并重新打开任何打开的命令提示符，以便在命令提示符下反映安装程序对环境所做的更改。 2. 请验证您是否已安装Go。

  1. 在Windows中，单击“开始”菜单。
  2. 在菜单的搜索框中，键入cmd，然后按Enter键。
  3. 在出现的“命令提示”窗口中，键入以下命令： $go version
  4. 确认该命令打印已安装的Go版本。

# 执行代码

安装配置完毕！访问入门教程，编写一些简单的Go代码。大约需要10分钟完成。
